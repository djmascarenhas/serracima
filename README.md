---

## 📖 Sobre o Projeto

**Serra Acima** é o resgate do nome histórico original de Chapada dos Guimarães (1750-1769). Este site preserva e compartilha 276 anos de história, desde a fundação da Missão Jesuítica de Santana até o moderno destino turístico que conhecemos hoje.

### 🎯 Missão

Conectar passado e presente através de narrativas históricas autênticas, documentos raros e roteiros turísticos culturais que valorizam a identidade única de Serra Acima.

---

## ✨ Características

- 🏛️ **História Completa** - 4 eras históricas documentadas com rigor acadêmico

## 🚀 Como Usar

### 1. Clonar o Repositório

```bash
git clone https://github.com/seu-usuario/serra-acima-site.git
cd serra-acima-site
```

### 2. Abrir Localmente

Basta abrir o arquivo `index.html` no navegador:

```bash
# Mac/Linux
open index.html

# Windows
start index.html
```

### 3. Ou usar um servidor local

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server

# Acesse: http://localhost:8000
```
